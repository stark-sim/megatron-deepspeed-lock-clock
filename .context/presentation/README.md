## 如何编译

### 要求
- LaTeX发行版 (TeX Live或MiKTeX)
- 必要的包: beamer, tikz, pgfplots, booktabs

### 编译命令

```bash
cd .context/presentation

# 编译PPT
make

# 或手动编译
pdflatex main.tex
pdflatex main.tex
mv main.pdf GPU_Energy_Optimization_20Percent_Savings.pdf
```

### 其他命令

```bash
make clean      # 清理辅助文件
make distclean  # 清理所有文件包括PDF
make view       # 在macOS上打开PDF
make slides     # 统计幻灯片数量
make lint       # 检查TODO/FIXME
```

## 核心数据

### 功耗节省指标

| 拓扑配置 | Baseline功率 | 优化后功率 | 节省比例 |
|----------|--------------|------------|----------|
| TP=1, PP=4, DP=4 | 3170.6 W | 2407.7 W | **24%** |
| TP=2, PP=2, DP=4 | 3686.1 W | 2477.3 W | **33%** |
| TP=4, PP=1, DP=4 | 3613.0 W | 2363.6 W | **35%** |

**平均节省: 30.7% (超过20%目标)**

### 预测准确性

| 环境 | 修复前MAPE | 修复后MAPE | 改进倍数 |
|------|-----------|-----------|----------|
| InfiniBand | 98.5% | 11.48% | 8.6× |

### 成本节省估算

以100卡集群训练30天为例:
- 节省电能: 6,480度
- 节省费用: **6.48万元**
