# 证据清单

这个文件只保留当前适合用于汇报总结的证据，并明确标注证据等级。

## 一、证据等级

- `A：本地工件可直接复核`
  - 当前 workspace 内仍存在原始文件，可直接打开
- `B：memory-bank 历史摘要`
  - 当前数字来自 `memory-bank/observability.md` 中已记录的历史 Zeus 摘要

## 二、方法与口径定义

| 结论 | 等级 | 来源 |
| --- | --- | --- |
| baseline 与 static 使用统一 launcher，差异由 `EXPERIMENT_MODE` 决定 | A | `docs/experiment-tracking.md` |
| `EXPERIMENT_MODE=baseline` 表示默认 GPU 行为 | A | `docs/experiment-tracking.md` |
| `EXPERIMENT_MODE=static` 加 `STATIC_CLOCK_MHZ` 表示锁到固定频点 | A | `docs/experiment-tracking.md` |
| Zeus 是当前项目 runtime / power / energy 的主观测来源 | A | `memory-bank/systemPatterns.md` 与 `memory-bank/observability.md` |

## 三、代表性案例 A：`TP=1, PP=4, DP=4`

证据等级：`B`

历史 baseline 摘要：

- baseline: `1316.7 s / 4174627.4 J / 3170.6 W / 0.392 tokens/J`

历史 static 摘要：

- `1252 MHz`: `1300.1 s / 3078625.8 J / 2367.9 W / 0.532 tokens/J`
- `1260 MHz`: `1287.9 s / 3067834.2 J / 2382.0 W / 0.534 tokens/J`
- `1267 MHz`: `1286.3 s / 3097007.8 J / 2407.7 W / 0.529 tokens/J`

相对 baseline 的变化：

| 频点 | 时长变化 | 平均功率变化 | 能耗变化 |
| --- | --- | --- | --- |
| `1252 MHz` | `-1.3%` | `-25.3%` | `-26.3%` |
| `1260 MHz` | `-2.2%` | `-24.9%` | `-26.5%` |
| `1267 MHz` | `-2.3%` | `-24.1%` | `-25.8%` |

这组结果适合支撑：

- `runtime 基本不变甚至略优，但平均功率下降约 25%`

## 四、代表性案例 B：`TP=2, PP=2, DP=4`

证据等级：`B`，同时有部分 `A` 级 static 本地工件支撑

历史 baseline 摘要：

- baseline: `1030.6 s / 3798903.8 J / 3686.1 W / 0.431 tokens/J`

历史 static 摘要：

- `1072 MHz`: `1053.6 s / 2475841.1 J / 2349.8 W`
- `1080 MHz`: `1046.1 s / 2471497.2 J / 2362.5 W`
- `1087 MHz`: `1040.0 s / 2482053.4 J / 2386.6 W`
- `1125 MHz`: `1006.1 s / 2492545.3 J / 2477.3 W`

相对 baseline 的变化：

| 频点 | 时长变化 | 平均功率变化 | 能耗变化 |
| --- | --- | --- | --- |
| `1072 MHz` | `+2.2%` | `-36.3%` | `-34.8%` |
| `1080 MHz` | `+1.5%` | `-35.9%` | `-34.9%` |
| `1087 MHz` | `+0.9%` | `-35.3%` | `-34.7%` |
| `1125 MHz` | `-2.4%` | `-32.8%` | `-34.4%` |

这组结果适合支撑：

- `在某些拓扑下，合适定频不仅不会明显拖慢训练，还能把平均功率压低到 30%+`

## 五、当前最适合在汇报里说的 headline

- 保守版本：
  - `合适的固定频点可带来约 20% 量级的平均功率下降，同时 runtime 基本不变。`
- 完整版本：
  - `在已完成的 V100 双机实验中，我们看到 25% 到 35%+ 的平均功率下降，而训练总时长通常保持在 baseline 附近。`
